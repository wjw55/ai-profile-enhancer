A Streamlit application that utilizes Google's Gemini AI to analyze resumes against specific job descriptions. It helps candidates understand their profile's strengths and identifies areas for improvement.

## Key Features

* **Smart Analysis:** Provides a match percentage between your resume and the target job role.
* **Format Support:** Accepts resumes in both `.pdf` and `.txt` formats.
* **Actionable Feedback:** Highlights missing skills and offers specific recommendations to improve your application.
* **Secure:** Built with security best practices for handling API keys.
